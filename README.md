# Internship Task 2

## Tech Stack
- Templating Engine: Nunjucks
- Bundler: Vite

## Folder Structure
- src/templates → Nunjucks templates
- src/assets → CSS, JS, images
- dist → Compiled output

## How to Run

1. Install dependencies:
   npm install

2. Start development server:
   npm run dev

3. Build final HTML:
   npm run build
